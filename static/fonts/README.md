# Fonts Directory

This directory contains all font files for CBT Mini School application.

## Files

- `inter/` - Inter font family
- `roboto/` - Roboto font family
- `material-icons/` - Material Icons font

## Optimization

Fonts are optimized for production using:
- Subsetting to include only used characters
- WOFF2 format for modern browsers
- Fallback formats for older browsers
