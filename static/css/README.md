# CSS Directory

This directory contains all stylesheets for the CBT Mini School application.

## Files

- `main.css` - Main application stylesheet
- `admin.css` - Admin-specific styles
- `student.css` - Student-specific styles
- `print.css` - Print-specific styles

## Build Process

CSS files are built from source files in `static/src/css/` using Tailwind CSS.

## Optimization

CSS files are optimized for production using:
- Minification
- Autoprefixer
- PurgeCSS for removing unused CSS
