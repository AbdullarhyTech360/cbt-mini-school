# JavaScript Directory

This directory contains all JavaScript files for CBT Mini School application.

## Files

- `main.js` - Main application JavaScript
- `admin/` - Admin-specific JavaScript files
- `student/` - Student-specific JavaScript files
- `components/` - Reusable JavaScript components
- `vendor/` - Third-party JavaScript libraries

## Build Process

JavaScript files are built from source files in `static/src/js/` using build tools.

## Optimization

JavaScript files are optimized for production using:
- Minification
- Bundling
- Tree shaking for removing unused code
