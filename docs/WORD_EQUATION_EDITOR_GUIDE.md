# Using Word Equation Editor for Math Questions

## ‚úÖ This is the EASIEST Way!

You don't need to learn LaTeX! Just use Word's built-in Equation Editor - it's visual and intuitive.

## How to Insert Equations in Word

### Method 1: Keyboard Shortcut (Fastest)
1. Press **`Alt + =`** (Alt and Equals key together)
2. The equation editor opens
3. Type your equation visually
4. Press **`Esc`** or click outside when done

### Method 2: Ribbon Menu
1. Go to **Insert** tab
2. Click **Equation** button
3. Type your equation
4. Click outside when done

## Step-by-Step: Creating a Complete Question

1. **Type the question label:**
   ```
   Question: 
   ```

2. **Type your question text:**
   ```
   Question: Simplify 
   ```

3. **Add the equation:**
   - Press `Alt + =`
   - Type your equation (e.g., `\sqrt{81}`)
   - Press Esc
   ```
   Question: Simplify ‚àö81
   ```

4. **Add question type:**
   ```
   Type: MCQ
   ```

5. **Add options:**
   ```
   Options:
   - 1
   - *9
   - 5
   - 81
   ```

6. **Done!** Save and upload.

- **Bracket** - For parentheses, brackets
- **Function** - For sin, cos, log, etc.

## Common Issues & Solutions

### Issue: Equation looks wrong after upload
**Solution:** Make sure you pressed Esc after finishing the equation

### Issue: Equation appears as plain text
**Solution:** Use Equation Editor (Alt + =), don't just type symbols

### Issue: Subscripts/superscripts not working
**Solution:** Use Equation Editor, type `_` for subscript, `^` for superscript
