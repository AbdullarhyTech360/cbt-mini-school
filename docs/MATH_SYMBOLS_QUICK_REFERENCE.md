# Mathematical Symbols Quick Reference

## How to Type Math in Your DOCX Questions

### Method 1: Direct Unicode Entry (Easiest)
Copy and paste these symbols directly into your Word document:

#### Numbers & Operations
| Symbol | Meaning | Example |
|--------|---------|---------|
| ² | Squared | x² = 25 |
| ³ | Cubed | x³ = 27 |
| √ | Square root | √16 = 4 |
| ± | Plus-minus | x = ±5 |
| × | Times | 3 × 4 = 12 |
| ÷ | Divide | 12 ÷ 3 = 4 |
| ≈ | Approximately | π ≈ 3.14 |
| ≠ | Not equal | 5 ≠ 3 |
| ≤ | Less or equal | x ≤ 10 |
| ≥ | Greater or equal | x ≥ 5 |
| ∞ | Infinity | lim x→∞ |

#### Greek Letters (Common in Math & Science)
| Symbol | Name | Usage |
|--------|------|-------|
| α | alpha | angle, coefficient |
| β | beta | angle, coefficient |
| γ | gamma | angle, gamma ray |
| δ | delta | change, difference |
| θ | theta | angle |
| λ | lambda | wavelength |
| μ | mu | micro, mean |
| π | pi | 3.14159... |
| σ | sigma | standard deviation |
| φ | phi | angle, golden ratio |
| ω | omega | angular velocity |

#### Calculus & Advanced
| Symbol | Meaning | Example |
|--------|---------|---------|
| ∑ | Sum | ∑(i=1 to n) |
| ∫ | Integral | ∫f(x)dx |
| ∂ | Partial derivative | ∂f/∂x |
| ∇ | Nabla/Del | ∇f |

#### Subscripts (Chemistry, Variables)
| Symbol | Example |
|--------|---------|
| ₀ | x₀ |
| ₁ | H₂O |
| ₂ | CO₂ |
| ₃ | CH₃ |
| ₄ | NH₄ |

### Method 2: LaTeX Notation (Most Powerful)

Wrap your math in dollar signs: `$your math here$`

#### Basic Examples
```
$x^2$           → x²
$x^{10}$        → x¹⁰
$\sqrt{x}$      → √x
$\sqrt[3]{x}$   → ³√x
$\frac{a}{b}$   → a/b (as fraction)
```

#### Equations
```
$x^2 + y^2 = z^2$
$E = mc^2$
$a^2 + b^2 = c^2$
$\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
```

#### Calculus
```
$\int_{0}^{1} x^2 dx$
$\sum_{i=1}^{n} i$
$\lim_{x \to \infty} f(x)$
$\frac{dy}{dx}$
```

#### Trigonometry
```
$\sin(\theta)$
$\cos(\theta)$
$\tan(\theta)$
$\sin^2(x) + \cos^2(x) = 1$
```

### Method 3: Word Equation Editor

1. In Word, press `Alt + =` to insert equation
2. Type your equation using Word's equation tools
3. The system will extract and convert it

## Common Question Patterns

### Chemistry
```
Question: What is the molecular formula for glucose?
Answer: C₆H₁₂O₆

Question: Balance: H₂ + O₂ → H₂O
Answer: 2H₂ + O₂ → 2H₂O
```

### Physics
```
Question: Calculate force using $F = ma$ where m = 5 kg, a = 2 m/s²
Answer: 10 N

Question: What is the formula for kinetic energy?
Answer: $KE = \frac{1}{2}mv^2$
```

### Mathematics
```
Question: Solve $x^2 - 5x + 6 = 0$
Answer: x = 2 or x = 3

Question: What is $\frac{3}{4} + \frac{1}{2}$?
Answer: $\frac{5}{4}$ or 1.25
```

## Tips for Best Results

### ✅ DO:
- Use LaTeX for complex formulas
- Use Unicode symbols for simple notation
- Test with a few questions first
- Keep equations readable
- Use consistent notation throughout

### ❌ DON'T:
- Mix too many notation styles in one question
- Use overly complex nested formulas
- Forget to mark correct answers with *
- Use images when text/symbols work fine
- Upload without testing first

## Keyboard Shortcuts (Windows)

### In Word:
- `Alt + =` - Insert equation
- `Ctrl + =` - Subscript
- `Ctrl + Shift + =` - Superscript

### Unicode Entry:
1. Type the code (e.g., 00B2 for ²)
2. Press `Alt + X`

Common codes:
- 00B2 → ²
- 00B3 → ³
- 221A → √
- 03C0 → π
- 03B1 → α
- 03B8 → θ
