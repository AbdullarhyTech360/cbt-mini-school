## Merge Rule Examples

### Example 1: Simple Two-Way Merge

**Input:**
```
CBT:  18/20
Exam: 55/60
```

**Merge Rule:**
```
Name: Final Exam
Components: [CBT, Exam]
Display As: final_exam
```

**Output:**
```
Final Exam: 73/80
```

---

### Example 2: Three-Way Merge

**Input:**
```
CA1: 9/10
CA2: 8/10
CA3: 9/10
```

**Merge Rule:**
```
Name: Total CA
Components: [CA1, CA2, CA3]
Display As: total_ca
```

**Output:**
```
Total CA: 26/30
```

---

### Example 3: Multiple Merges

**Input:**
```
CA1:  9/10
CA2:  8/10
CBT:  18/20
Exam: 55/60
```

**Merge Rules:**
```
Rule 1:
  Name: Mid-term
  Components: [CA1, CA2]
  Display As: midterm

Rule 2:
  Name: Final Exam
  Components: [CBT, Exam]
  Display As: final_exam
```

**Output:**
```
Mid-term:   17/20
Final Exam: 73/80
```

## Quick Reference: Common Tasks

### Task 1: Create Basic Config
```
1. Go to /reports/config
2. Click "New Configuration"
3. Name: "Basic Report"
4. Select term
5. Check all assessments
6. Save
```

### Task 2: Merge CBT + Exam
```
1. Create new config
2. Add merge rule:
   - Name: Final Exam
   - Components: CBT, Exam
   - Display: final_exam
3. Active: final_exam only
4. Save
```

### Task 3: Generate Class Reports
```
1. Go to /reports/generate
2. Select term and class
3. Click "Load Students"
4. Click "Download All"
5. Wait for PDF
```
