## Troubleshooting

### Problem: Exam dropdown is disabled
**Solution**: Select a student first

### Problem: No exams showing
**Solution**: Student hasn't completed any exams yet

### Problem: Reset button is disabled
**Solution**: Select both student and exam

### Problem: Error message appears
**Solution**: 
1. Check internet connection
2. Refresh the page
3. Try again
4. Contact support if persists

### Problem: Success message but student can't access exam
**Solution**:
1. Ask student to refresh their page
2. Check student is enrolled in subject
3. Verify exam is still active
4. Contact support

## Frequently Asked Questions

**Q: Can I reset multiple exams at once?**
A: No, reset one exam at a time for safety.

**Q: Can I undo a reset?**
A: No, resets are permanent.

**Q: Will the student be notified?**
A: No, you must inform them manually.

**Q: Can students see their old score?**
A: No, it's completely deleted.

**Q: How long does reset take?**
A: Usually 2-3 seconds.

**Q: Can I reset an exam that wasn't completed?**
A: No, only completed exams can be reset.

**Q: What if I select the wrong exam?**
A: Use the confirmation modal to verify before confirming.

**Q: Can students reset their own exams?**
A: No, only admins can reset exams.

**Q: Is there a limit to resets?**
A: No technical limit, but use responsibly.

**Q: What happens to the exam statistics?**
A: Previous attempt is removed from statistics.
