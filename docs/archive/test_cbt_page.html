CBT Test Page Diagnostic

Instructions:

  1. Make sure you're accessing the correct URL: /student/exam/<exam_id>/start
  2. NOT the API endpoint: /student/exam/<exam_id>/questions
  3. Open browser DevTools (F12) and check the Console tab for errors
  4. Check the Network tab to see if JavaScript files are loading

Common Issues:

  • Seeing raw JSON? You're on the API endpoint, not the test page
  • JavaScript not loading? Check browser console for errors
  • MathJax not rendering? Wait a few seconds for MathJax to load

Quick Fix:

Clear your browser cache (Ctrl+Shift+Delete) and hard refresh (Ctrl+F5)
