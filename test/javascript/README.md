# JavaScript Tests

This directory contains JavaScript unit tests and integration tests for the CBT Mini School project.

## Test Files

- `test_client_pdf_generation.js` - Tests for client-side PDF generation
- `test_enhanced_styling.js` - Tests for enhanced PDF styling
- `test_error_handling.js` - Tests for error handling in PDF generation
- `test_improved_pdf.js` - Tests for improved PDF generation
- `test_modern_pdf.js` - Tests for modern PDF generation
- `comprehensive_nan_test.js` - Comprehensive tests for NaN handling
- `logging_test.js` - Tests for logging functionality
- `nan_test.js` - Tests for NaN handling

## Running Tests

```bash
# Run all JavaScript tests
npm test

# Run specific test file
npm test test/javascript/test_client_pdf_generation.js

# Run tests with coverage
npm run test:coverage
```

## Test Configuration

Test configuration is defined in `package.json` in the project root.

## Test Framework

Tests use the Jest testing framework.
